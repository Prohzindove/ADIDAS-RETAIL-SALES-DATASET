# ADIDAS-RETAIL-SALES-DATASET
Adidas sales data across six retail outlets was analysed. The dataset was first loaded into a data bricks workspace where it was checked to gain an understanding of its contents. The data contained null values with the first column having all null values. This resulted in the deletion of all the data while trying to remove the null values. The data was resored using a timestamp. The first column was then removed and the remaining null values were removed. Additionally, the columns were renamed to give them meaning, for example, column _01 was renamed to Retailer. This ensured that the data was claen and meaningful before coding.
